Chapter 2

安裝 Git 與設定環境

這一章先把工具準備好，讓你後面在練習 commit、push 與 GitHub Pages 時不會被環境問題卡住。

安裝 Git

先到 Git 官方網站下載安裝程式，完成安裝後打開終端機，輸入以下指令確認是否成功：

git --version

設定 Git 身分

第一次使用 Git 時，建議先設定使用者名稱與電子郵件，這些資訊會寫入 commit 紀錄中。

git config --global user.name "Your Name"
git config --global user.email "you@example.com"
提醒

這裡填寫的資訊最好和你的 GitHub 帳號一致，後續查看 commit 紀錄會更清楚。

準備 VS Code

安裝 VS Code 後，你可以直接使用內建的 Source Control 功能來查看變更、建立 commit、同步遠端。

  • 安裝 VS Code
  • 打開專案資料夾
  • 熟悉左側的 Source Control 面板

驗證環境

在開始下一章之前，確認你已經能做到以下幾件事：

  • 終端機可正確執行 `git --version`
  • 已設定 `user.name` 與 `user.email`
  • VS Code 可正常開啟專案資料夾
