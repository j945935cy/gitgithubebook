Lab 1

建立第一個 GitHub Repo

這個練習會帶你完成最小可行流程：建立 repo、clone 到本機、新增檔案、commit、push。

練習目標

  • 建立一個新的 GitHub repository
  • 把 repository clone 到本機
  • 完成第一次 commit 與 push

先備條件

  • 已擁有 GitHub 帳號
  • 已安裝 Git 與 VS Code
  • 已完成基本 Git 設定

操作步驟

1

在 GitHub 建立新 repository

命名可以先用簡單的名稱，例如 `my-first-project`。

2

clone 到本機

git clone https://github.com/your-account/my-first-project.git
3

新增一個 `index.html`

在資料夾裡建立最基本的首頁檔案。

4

建立第一個 commit

git add .
git commit -m "initial commit"
5

push 到 GitHub

git push

VS Code 操作對照

  • 打開專案資料夾
  • 進入 Source Control
  • 輸入 commit message
  • 點擊 Commit 與 Sync / Push

成功條件

  • GitHub 上可以看到新檔案
  • 本機與遠端 repository 已同步
  • 你知道 `clone`、`commit`、`push` 分別在做什麼

常見錯誤

`push` 失敗

先確認你是否有正確登入 GitHub、遠端網址是否正確，以及遠端是否已有你本機沒有的內容。
