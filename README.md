# Git Start Book

Git Start Book 是一個以 `Git`、`GitHub`、`VS Code` 為主題的靜態教學網站，整體視覺方向參考 Apple 官網，並加入低飽和淡綠色點綴。

## Current Scope

- `index.html` 首頁
- `toc.html` 課程目錄
- `chapters/ch01.html`
- `chapters/ch02.html`
- `labs/lab01-student-repo.html`
- `style/main.css`
- `js/main.js`

## Local Preview

直接用瀏覽器開啟 `index.html` 即可預覽，或使用任何靜態伺服器工具啟動本機網站。

## Deployment

此專案適合直接部署到 GitHub Pages。

基本流程：

1. 建立 GitHub repository
2. 將專案推送到 GitHub
3. 在 repository 設定中開啟 GitHub Pages
4. 將發布來源設為主分支

## Design Notes

- 黑白灰為主體，淡綠為點綴
- 保持大量留白與穩定閱讀節奏
- 內容優先，動畫只作輔助
